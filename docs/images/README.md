# Website Screenshots

This directory contains screenshots of the Unified Lightning Wallet website for display in the README.

Please add the following screenshots:

1. **hero.png** - Landing page hero section
2. **wallet.png** - Wallet interface demo page
3. **features.png** - Features section showcase
4. **downloads.png** - Multi-platform download section

## Taking Screenshots

- Resolution: 1920x1080 or higher
- Format: PNG (preferred) or JPG
- Crop to show relevant content
- Ensure good lighting/contrast

Once you add screenshots, they will appear in the main README.md file.
